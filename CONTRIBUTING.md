# Guia de Contribuição para o AgentCore

Ficamos felizes com o seu interesse em contribuir para o AgentCore! Este guia estabelece as diretrizes para garantir que o projeto continue evoluindo de forma organizada e consistente.

## Código de Conduta

Esperamos que todos os contribuidores sigam um código de conduta que promova um ambiente aberto, acolhedor e respeitoso. Por favor, trate todos com respeito e profissionalismo.

## Como Contribuir

Você pode contribuir de várias formas: reportando bugs, sugerindo novas funcionalidades, melhorando a documentação ou enviando código.

### Reportando Bugs

Se você encontrar um bug, por favor, crie uma "Issue" no repositório do GitHub. No seu reporte, inclua:

-   **Título claro e descritivo.**
-   **Descrição detalhada do problema:** o que aconteceu e o que você esperava que acontecesse.
-   **Passos para reproduzir o bug:** forneça um exemplo de código ou um guia passo a passo.
-   **Informações do seu ambiente:** versão do Node.js, sistema operacional, etc.

### Sugerindo Melhorias

Para sugerir uma nova funcionalidade ou uma melhoria, crie uma "Issue" com a tag `enhancement`. Descreva sua ideia com o máximo de detalhes possível, explicando o caso de uso e por que ela seria útil para o projeto.

### Enviando Pull Requests

Para contribuir com código ou documentação, siga os passos abaixo:

1.  **Faça um Fork do Repositório:** Crie uma cópia do projeto na sua conta do GitHub.
2.  **Crie uma Nova Branch:** A partir da branch `main`, crie uma nova branch para suas alterações. Use um nome descritivo (ex: `feature/nova-simulacao` ou `fix/bug-no-script`).
    ```sh
    git checkout -b feature/minha-feature
    ```
3.  **Desenvolva suas Alterações:** Faça as modificações necessárias no código.
4.  **Faça o Commit das Alterações:** Use mensagens de commit claras e informativas. Recomendamos seguir o padrão [Conventional Commits](https://www.conventionalcommits.org/en/v1.0.0/).
    ```sh
    git commit -m "feat: Adiciona novo fluxo de saudação"
    ```
5.  **Envie para o seu Fork:**
    ```sh
    git push origin feature/minha-feature
    ```
6.  **Abra um Pull Request (PR):** Acesse o repositório original e crie um Pull Request da sua branch para a branch `main`. No PR, descreva detalhadamente as mudanças que você fez e por quê.

Aguarde a revisão do seu PR. Os mantenedores do projeto irão analisar suas alterações e fornecer feedback.

Obrigado por contribuir para o futuro do AgentCore!
